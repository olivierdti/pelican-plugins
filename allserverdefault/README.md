# All-servers default

Makes the `All servers` option the default one in the server panel.

## Features

- Automatically selects "All servers" as the default view when accessing the server panel
- Simple and lightweight plugin with no configuration required

## Installation

1. Download the plugin ZIP file from the [Releases](https://github.com/olivierdti/pelican-plugins/releases) page
2. Go to your Pelican Panel admin area
3. Navigate to the Plugins page
4. Click on "Import file"
5. Select the downloaded ZIP file
6. Click "Import"

## Requirements

- Pelican Panel v1.0+
