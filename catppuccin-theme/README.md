# Catppuccin Theme

A cute, little theme for Pelican panel inspired by the Catppuccin color palette.

## Features

- Beautiful Catppuccin color scheme
- Consistent styling across all panels (admin, app, server)
- Optimized for readability and visual comfort

## Installation

1. Download the theme ZIP file from the [Releases](https://github.com/olivierdti/pelican-plugins/releases) page
2. Go to your Pelican Panel admin area
3. Navigate to the Plugins page
4. Click on "Import file"
5. Select the downloaded ZIP file
6. Click "Import"

## Requirements

- Pelican Panel v1.0+
